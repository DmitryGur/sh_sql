## 📌 Цели и задачи проекта

Проект представляет собой анализ датасета с вакансиями HeadHunter по городам РФ.  
Цель — выявить ключевые факторы, влияющие на рынок труда и ожидания работодателей, и подготовить данные для построения прогнозных моделей.

**Основные задачи:**
- Знакомство с данными;
- Предварительный анализ данных;
- Детальный анализ вакансий;
- Анализ работодателей;
- Предметный анализ.
